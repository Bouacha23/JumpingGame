• Play Game

    Welcome to Run Branko Run the goal of the game is to help Branko run away from bugs in the code and hopefully hide some so he doesnt notice them

    You need to get 1000 points to win. If you run out of points you lose.

    Controls

    Jump: Space

    Mute: m

    Pause: p | Esc

    Increase Volume v

    Decrease Volume b
